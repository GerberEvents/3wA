Au cours de cette formation nous allons avoir 2 matières :

L'intégration

le HTML

Le HTML sert à créer le contenu du site : elle va contenir uniquement la partie textuelle du site.

le CSS

Le CSS va nouspermettre d'ajouter la couche graphique à notre site : les couleur, les formes, le fond, etc.

Le développement

le Javascript :

javascript est un langage de programmation de script employé pour rendre une page interactive, le comportement de la page et son apparence pourra varier en fonction de certains paramétres.

Le PHP

Le PHP est un langage de programation libre, utilisé pour généré des parges web dynamiques, qui sont donc modifiés en fonction de certaines information (heure, nom de l'utilisateur, formulaire rempli par l'utilisateur etc).

MySQL

MySQL est un systéme de gestion de bases de données, il permet d'enregistrer des informations rempli par l'utilisateur (exemple:formulaire)
